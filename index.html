    How to collaborate

    Using Git

    This notes are saved into a Git repository. If you know how to use Git, feel free to use it:

    git clone git@github.com:michogar/modeling.git
cd modeling

    make changes and`

    git commit -a
git push

    Without using Git

    A simple way to collaborate is downloading notes from here. Then you can modify whatever you want and after that, send the apuntes.md file to michogar at gmail dot com with the subject [MODELING] in the mail.
